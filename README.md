# gomigraches

The utility that created for easily database and their tables rolling up

## CLI usage

### Build util

```bash
# go to cli folder
cd ./cmd/cli

# build up util
go build

# run it
./cli table_to_rollup postgres superuser_role_user 

# or just run util to see help information
./cli
```
